# Bank Hisoblari Boshqaruv Tizimi

Ushbu amaliy loyiha mijozlarning bank hisob raqamlarini yaratish, ularning balanslarini boshqarish hamda hisoblararo pul o'tkazmalarini amalga oshirishga mo'ljallangan console dasturidir.

## Imkoniyatlari:
- `och <ism>` - Yangi mijoz hisobini ochish.
- `qoy <raqam> <summa>` - Balansni to'ldirish.
- `balans <raqam>` - Hisob balansini tekshirish.
- `yech <raqam> <summa>` - Hisobdan mablag' yechish.
- `otkaz <dan> <ga> <summa>` - Ikki hisob orasida xavfsiz pul o'tkazish.
- `tarix <raqam>` - Tranzaksiyalar tarixini ko'rish.
- `hisobot` - Umumiy bank tizimi statistikasini chiqarish.

## Ishga tushirish:
Dasturni ishga tushirish uchun quyidagi buyruqni bajaring:
```bash
python main.py
```
